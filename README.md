# SIR Sazonal Model

Modelo epidemiológico SIR com transmissão sazonal.

## Descrição

Simulação da dinâmica de uma doença respiratória utilizando modelo determinístico com forçamento periódico na taxa de transmissão.

## Requisitos

- Python 3.8+
- numpy
- matplotlib

## Instalação

pip install numpy matplotlib

## Execução

python sir_sazonal.py

## Parâmetros editáveis

- beta0
- gamma
- alpha
- população
- tempo de simulação
